# Career card resume

Interactive “pack opening” portfolio: branded pack rip, collectible-style career cards, PDF résumé link.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown (usually http://localhost:5173).

## Build

```bash
npm run build
```

Output is in `dist/` (used for deployment).

## Share with recruiters

Put the code on **GitHub**, then deploy to **Vercel** or **Netlify** (both have free tiers). Connect your GitHub account, import this repo, and use the default settings for a Vite app. You’ll get a public URL like `https://your-project.vercel.app` to put on your LinkedIn, email signature, or applications.
